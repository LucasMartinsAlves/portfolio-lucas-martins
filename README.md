# Portfólio acadêmico

Projeto de portfólio pessoal para a atividade prática de Fundamentos da Programação Web.

## Arquivos

- `index.html`: estrutura da página e conteúdo das seções.
- `style.css`: estilos visuais e responsividade.
- `script.js`: menu mobile, tema claro/escuro e validação do formulário.
- `assets/profile-card.svg`: imagem visual usada no início da página.

## O que trocar antes de entregar

Procure por estes textos no `index.html` e ajuste com seus dados reais:

- `Lukas`
- `estudante em desenvolvimento web`
- texto da seção `Sobre mim`
- informações de `Formação`
- links e descrições da seção `Portfólio`
- ano, se necessário

No `index.html`, quando tiver o link do seu GitHub ou LinkedIn, você pode incluir esses links na seção de contato ou no portfólio.

## Como publicar no GitHub Pages

1. Crie um repositório público no GitHub.
2. Envie estes arquivos para o repositório.
3. No GitHub, abra `Settings`.
4. Entre em `Pages`.
5. Em `Build and deployment`, selecione `Deploy from a branch`.
6. Escolha a branch `main` e a pasta `/root`.
7. Salve e aguarde o GitHub gerar o link.

O link final costuma ficar neste formato:

`https://seu-usuario.github.io/nome-do-repositorio/`

## Entrega no modelo da Uninter

No PDF final, inclua:

- link do site publicado;
- link do repositório GitHub público;
- print da estrutura de pastas;
- prints das seções Sobre mim, Formação, Portfólio e Contato com a URL publicada visível.
